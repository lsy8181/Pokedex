## 📖 Project Description

해당 사이트는 **1세대 포켓몬**을 볼 수 있는 도감을 만들었습니다. 포켓몬 카드를 클릭하면 해당 포켓몬의 상세 내용을 볼 수 있습니다.

### 🛠️ 사용 기술 스택

- **React**
- **TypeScript**
- **Tailwind CSS**
- **Framer Motion API**
- **포켓몬 API**

## 🎨 Features

- **포켓몬 도감**: 1세대 포켓몬의 리스트를 볼 수 있습니다.
- **포켓몬 상세 정보**: 포켓몬 카드를 클릭하면 상세 정보를 확인할 수 있습니다.
- **애니메이션 효과**: Framer Motion을 사용한 부드러운 애니메이션 효과가 적용되었습니다.
